# 🐟 Fish Detection API - Render FREE Tier Quick Card

## 3-Step Deployment

### Step 1️⃣: Git LFS Setup (2 min)
```powershell
cd f:\fish model backend
.\setup-render.bat
```

### Step 2️⃣: Push to GitHub (5 min)
```powershell
git add .
git commit -m "Ready for Render FREE deployment"
git remote add origin https://github.com/YOUR_USERNAME/fish-detection-api.git
git push -u origin main
```

### Step 3️⃣: Deploy on Render (15 min)
1. https://dashboard.render.com
2. **New** → **Web Service**
3. Select repo → **Plan: FREE** ⚠️
4. Create & wait

## 🆘 Troubleshooting

| Problem | Fix |
|---------|-----|
| Models missing | `git lfs ls-files` should show them |
| Out of memory | Reduce WORKERS to 1 in render.yaml |
| First request slow | That's cold start - wait 50 sec |
| Build error | Check Render logs |
